Hem fet una aplicació utilitzant la api jikan, que conté dades de series d'animació japoneses, en el nostre cas l'api et mostra els 50 animes amb millor valoració, separats per temporades.
Quan iniciem l'aplicació, el primer que es mostra son els animes en 2 columnes, ordenats per valoració, a sota es veuen els resultats totals i a dalt està la search bar:
