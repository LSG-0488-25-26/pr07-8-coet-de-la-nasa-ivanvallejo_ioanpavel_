Hem fet una aplicació utilitzant la api jikan, que conté dades de series d'animació japoneses, en el nostre cas l'api et mostra els 50 animes amb millor valoració, separats per temporades.
Quan iniciem l'aplicació, el primer que es mostra son els animes en 2 columnes, ordenats per valoració, a sota es veuen els resultats totals i a dalt està la search bar

<img width="402" height="887" alt="image" src="https://github.com/user-attachments/assets/81a99f00-d7ee-473d-a9e6-b0da4e0ba6d9" />

Si cliquem sobre una serie, es veure una vista més detallada, amb dades com el nom, valoració, sinopsis, número de capítols etc...

<img width="360" height="785" alt="image" src="https://github.com/user-attachments/assets/a2b1b80f-2b4a-425d-b83d-7f1a8a4ac48b" />
